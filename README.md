# バス出発時刻案内

最寄りのバス停の出発時刻や、次のバスまでの残り時間をリアルタイムで確認できるWebアプリケーションです。

## 機能

*   **次のバスの案内**: 現在時刻に基づき、次のバスの出発時刻と残り時間を表示します。
*   **ダイヤ自動判定**: 平日・土日祝日を自動で判定し、適切な時刻表を表示します（日本の祝日判定ライブラリを使用）。
*   **手動切り替え**: 「平日ダイヤ」「休日ダイヤ」ボタンで、手動で表示する時刻表を切り替えることができます。
*   **時刻表一覧**: 全ての出発時刻を一覧で確認できます。出発済みの便はグレーアウトされます。
*   **自動更新**: 画面は30秒ごとに自動更新され、常に最新の情報を表示します。

## 使い方

1.  リポジトリをクローンまたはダウンロードします。
2.  `index.html` をWebブラウザで開きます。
3.  自動的に現在の曜日・時刻に合わせたバスの運行状況が表示されます。

## ファイル構成

*   `index.html`: アプリケーションのメイン画面
*   `script.js`: アプリケーションのロジック（時刻表の読み込み、時刻計算、表示更新など）
*   `timetable.json`: バス停ごとの時刻表データ（JSON形式）
*   `style.css`: デザイン・レイアウト定義

## データ構造 (`timetable.json`)

時刻表データは以下の構造で管理されています。

```json
{
  "平日": {
    "バス停名": {
      "行き先・ルート名": ["06:28", "06:59", ...]
    }
  },
  "休日": {
    ...
  }
}
```

## 使用技術

*   HTML5
*   CSS3
*   JavaScript (Vanilla JS)
